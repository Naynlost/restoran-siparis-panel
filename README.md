# Restoran Sipariş Paneli

Bu proje, bir restoranın menü yönetimi ve sipariş işlemlerini gerçekleştirmek için geliştirilmiş basit bir web uygulamasıdır.

https://restoran-paneli.web.app/

## Kullanılan Teknolojiler
* **Frontend:** React.js (Vite)
* **Styling:** Bootstrap 5 & React-Icons
* **Backend/Database:** Firebase Firestore
* **Deployment:** Firebase Hosting

## Özellikler
* Menüdeki ürünleri görüntüleme
* Menüye yeni ürün ekleme, silme ve fiyat güncelleme (CRUD)
* Sepete ürün ekleme ve miktar yönetimi
* Sipariş özeti ve onay süreci
* Responsive tasarım (Mobil uyumlu)
